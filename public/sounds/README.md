# Notification Sounds

Esta pasta deve conter os arquivos de áudio para notificações:

## Arquivos Necessários

1. **ringtone.mp3** - Som de chamada recebida (toca em loop até aceitar/recusar)
2. **message.mp3** - Som de mensagem nova
3. **notification.mp3** - Som genérico para outros tipos de notificação

## Onde encontrar sons gratuitos

- [Zapsplat](https://www.zapsplat.com/) - Sons gratuitos (requer conta)
- [Freesound](https://freesound.org/) - Sons Creative Commons
- [Pixabay](https://pixabay.com/sound-effects/) - Sons livres de direitos
- [Mixkit](https://mixkit.co/free-sound-effects/) - Sons gratuitos

## Recomendações

- **ringtone.mp3**: Som suave e repetitivo (5-10 segundos é ideal para loop)
- **message.mp3**: Som curto e discreto (1-2 segundos - tipo "pop" ou "ding")
- **notification.mp3**: Som médio (2-3 segundos - tipo "bell" ou "chime")

## Formato

- Formato: MP3 (melhor compatibilidade cross-browser)
- Bitrate: 128kbps é suficiente
- Sample Rate: 44.1kHz
- Tamanho: Manter arquivos pequenos (<100KB cada)

## Teste Rápido

Você pode usar sons do sistema temporariamente ou criar arquivos MP3 vazios inicialmente. O sistema não vai travar se os arquivos não existirem, apenas não vai tocar som.

## Criação de Sons Temporários com Audacity

Se tiver Audacity instalado:
1. Generate > Tone > Sine wave (440Hz, 1 segundo)
2. Export as MP3
3. Renomear para o arquivo desejado
